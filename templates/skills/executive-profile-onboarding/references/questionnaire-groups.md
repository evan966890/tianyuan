# 问卷分组

## 第1组：基础信息

- 姓名（必填，文本）
- 希望 AI 怎么称呼你（必填，文本）
- 职位（可跳过，默认“总经理”）
- 所在部门（可跳过）
- 主要业务领域（可跳过）

## 第2组：沟通偏好

- 2a 你希望 AI 怎么跟你说话？单选
  - 直接给结论
  - 结构化展开
  - 叙事式
  - 数据驱动
- 2b 默认回复长度？单选
  - 极简（3-5行）
  - 精炼（5-12行）
  - 适中（12-20行）
  - 详尽（按需展开）
- 2c 偏好的信息格式？多选
  - 要点列表
  - 精炼段落
  - 对比表格
  - 选项卡（A/B/C）
- 2d 最反感的 AI 行为？多选
  - 正确但无用的废话
  - 没有行动建议的分析
  - 编造数据或进度
  - 过度热情/谄媚
  - 把噪音当情报
  - 冗长的背景铺垫
  - 模棱两可不敢给建议
  - 过度使用术语
- 2d-other 还有其他讨厌的？可选文本
- 2e 语言偏好，默认：`中文为主，专业术语保留 English`

## 第3组：决策风格

- 3a 常用思维框架？多选
  - MECE分析
  - 5-Why
  - OKR
  - SWOT
  - 第一性原理
  - 10x思维
  - 逆向工作法
  - OODA循环
  - 博弈论思维
  - 贝叶斯更新
  - Jobs-to-be-Done
  - 飞轮效应
- 3b 决策习惯或原则，多行文本

## 第4组：关键人物

动态列表，每行：

- 姓名/称呼
- 角色
- 备注（可空）

## 第5组：工作重点

动态标签列表，建议 3-5 项。

## 第6组：敏感事项

- 敏感业务 / 合作 / 话题，多行文本

## 第7组：知识偏好与推送

- 7a 知识兴趣领域，多选
  - AI/大模型
  - 商业战略
  - 组织管理
  - 产品设计
  - 技术架构
  - 宏观经济
  - 投资理财
  - 历史哲学
  - 心理学
  - 创新方法论
  - 行业研究
  - 领导力
- 7a-other 其他关注方向，可选文本
- 7b 推送模式，单选
  - 主动推送型
  - 按需搜索型
  - 混合型
- 7c 推送频率，单选，仅在 7b=主动推送型/混合型 时显示
  - 每天
  - 每两天
  - 每周
- 7d 推送时间，单选，仅在 7b=主动推送型/混合型 时显示
  - 早上（8:00-9:00）
  - 中午（12:00-13:00）
  - 晚上（20:00-21:00）
- 7e 每次推送条数，单选，仅在 7b=主动推送型/混合型 时显示
  - 精简（3条）
  - 适中（5条）
  - 丰富（7条）
- 7f 信息语言偏好，单选
  - 中文为主
  - 英文为主
  - 中英都要
- 7g 阅读 / 学习习惯，多行文本

### 对话式补充规则

- 第 7 组在聊天 onboarding 中默认至少 3 轮，不要一轮结束
- 第 1 轮只做主题粗分类
- 第 2 轮细化维度 / 深度 / 语言
- 第 3 轮确定推送模式 / 频率 / 条数
- 如果用户只回一个短词，AI 需要先给出 2-5 个最可能的解释让用户选
- 用户回复应尽量支持最短格式：`1`、`1 3`、`2 B`、`跳过`

## 第8组：预览与导出

向用户展示：

- `USER.md`
- `MEMORY.md`
- `radar/INTERESTS.md`（仅主动推送型 / 混合型）

并提供复制用的 Markdown 原文。
