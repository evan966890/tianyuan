# Triage Patterns

## 先判断语音属于哪类

1. 决策判断
2. 信息整理
3. 行动跟进
4. 个人记录

## 缩短用户输入的方法

- 不问“你想让我做什么”，改问“更像 1/2/3 哪种”
- 不问“详细背景”，改问“这是今天要处理，还是先存着”
- 不问“你具体要什么格式”，改问“要结论 / 清单 / 归档”

## 何时需要第二轮确认

- transcript 明显缺词
- 一个语音里混了两个任务
- 用户说法像自言自语，目标还没成型

## 禁忌

- 不把整段 transcript 原样搬给用户
- 不因为 transcript 口语化就装作没听懂
- 不连续追问 3 个开放式问题
