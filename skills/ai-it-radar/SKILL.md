# AI/IT 雷达

每天筛选全球 AI、芯片、消费电子领域最有价值的 5-7 条新闻。

## 触发

- 定时任务
- 心跳检查
- 用户主动询问

## 输出格式

```text
AI/IT 日报 | YYYY-MM-DD

1. [公司名] 一句话摘要（为什么重要）
2. [公司名] 一句话摘要（为什么重要）
```

## 信息源优先级

1. 官方公告 / 新闻稿
2. 权威科技媒体
3. 社交媒体一手消息

## 筛选标准

- 过去 24 小时内
- 对行业有实质影响
- 与用户关注领域直接相关
- 同一事件去重
